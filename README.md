## Tips for sharing

- **Always send the invite code separately or in a bold/quoted format** — easy to miss in a long message
- **Mention "Add to Home Screen"** — most friends won't know to do this, and it dramatically improves the mobile experience
- **Don't share the invite code publicly** (Twitter, public Discord, etc.) — anyone with the code can sign up. Rotate it in Streamlit secrets if it leaks
- **Test the link yourself first** in an incognito window to make sure signup works before blasting friends
